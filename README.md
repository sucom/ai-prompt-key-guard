*AI Prompt Key Guard* is a single-purpose browser extension to prevent accidental, half-baked AI prompts being sent to AI. Default "Enter" key creates a new line; use Ctrl/Alt/Cmd + Enter to submit on Gemini, Claude, Grok, ChatGPT and more.

1. Install extension from Chrome Web Store
2. Pin extension in the toolbar for easy access.
3. Visit supported AI sites.
4. Now, compose detailed prompt including newlines using simple Enter key. When you're ready, press Ctrl+Enter to submit your prompt.
5. Use extension's settings to change submit key shortcut.
6. No information is collected or transferred.

## Key Features
* Safe Typing: The Enter key will insert a new line in the prompt box.
* Custom Submit Shortcut: Choose between Ctrl + Enter, Alt (Option) + Enter, or Cmd + Enter to actually submit your prompt.
* Quick Toggle: A simple on/off switch in the extension popup lets you disable the key guard temporarily without uninstalling.
* Visual Indicator: The extension icon turns gray when disabled, so you always know your status at a glance.

## Supported AI Platforms on Chrome
AI Prompt Key Guard is tailor-made to support the complex text editors of the most popular AI tools:
* Google Gemini
* Claude
* Grok
* ChatGPT
* DeepSeek
* Microsoft Copilot

## Limitation (Known Issue)
* Though this extension can be installed on Microsoft Edge, it doesn't work for Microsoft Copilot!

## Privacy First
This extension requires zero permissions to read your browsing history or access external servers. Your shortcut preference and toggle state are saved entirely locally on your device. It simply observes your keystrokes inside the prompt box to prevent accidental submissions.
